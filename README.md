# 簡易電子琴

呼叫 Win API 製作的 25 鍵虛擬電子琴，透過加入「移調」與「樂器音色選擇」兩個主要功能模擬真實電子琴。


## 核心功能

### 1. 發聲效果

* **低延遲與和弦支援**：基於 Beep 的單音發聲修改，盡量達到觸覺與聽覺同步的即時發聲體驗;同時支援同時按下多個按鍵彈奏和弦，讓使用者感受貼近真實樂器的演奏體驗。
  
* **128 種音色**：內建 128 種標準音色（編號 0-127），涵蓋大鋼琴、木吉他、管弦樂團至現代電子合成音效，使用者可透過下拉選單或直接輸入編號快速切換所需樂器。
  
### 2. 彈奏與控制

* **完整音階覆蓋**：雖然畫面呈現為 25 鍵（雙八度），但使用者可以透過向上或向下平移音域突破畫面限制，實現接近真實大鋼琴的 88 鍵完整音階的效果。
  
* **雙模式控制**：支援「滑鼠點擊」與「電腦鍵盤」雙模式彈奏，使用者不僅能透過鍵盤敲擊發聲，還能利用快捷鍵快速完成升降調切換與樂器選擇，大幅提升演奏連貫性。

### 3. 視覺設計

* **視覺回饋**：當使用者透過電腦鍵盤彈奏或滑鼠點擊時，畫面上對應的黑白琴鍵會瞬間變色亮起，讓使用者能清楚知道自己目前按下了哪些音符，對於練習與確認音準具幫助。
  
* **視窗自動縮放**：當使用者改變視窗大小時，所有的琴鍵與控制項皆會依比例自動、精準地縮放，確保在任何螢幕解析度下，都能維持最完美的視覺比例與絕佳的操作手感。

1.  **彈奏方式**：
   
    * **滑鼠點擊**：直接點擊畫面上的黑白鍵即可發聲。
      
    * **鍵盤彈奏**：
      
      * 第一八度/低音：使用下排字母「Z, X, C, V, B, N, M」對應白鍵「Do, Re, Mi, Fa, So, La, Si」，「S, D, G, H, J」對應黑鍵「Do#, Re#, Fa#, So#, La#」。
        
      * 第二八度/高音：使用上排字母「Q, W, E, R, T, Y, U, I」對應白鍵「Do, Re, Mi, Fa, So, La, Si, Do」，數字鍵「2, 3, 5, 6, 7」對應黑鍵「Do#, Re#, Fa#, So#, La#」。

2.  **切換樂器音色**：
   
    * **下拉選單查找**：點擊選單展開 128 種樂器清單，已依樂器家族（如：鋼琴類、吉他類）分門別類。

    * **快速代碼輸入**：可直接在選單文字框內輸入 0~127 之間的數字並按下 Enter 鍵，系統會自動匹配對應的樂器名稱並切換。
      
    * **秒切快捷鍵**：彈奏中可隨時按下 F1 (鋼琴)、F2 (吉他)、F3 (小提琴)、F4 (小號) 進行快速音色變換。

3.  **音域移調**：
   
    * **介面滑動**：預設第一八度為 C4，拖曳畫面上的移調滑動條 (TrackBar)，可選擇第一八度為 C1~C6。
      
    * **上下方向鍵控制**：彈奏過程中，可直接按下電腦鍵盤的方向鍵 `Up` 或 `Down` 即時切換高低八度，介面滑桿將自動同步更新。

4.  **提醒視窗**：
   
    * 當使用者在音色選單內輸入非數字（如英文字母）或超出 0~127 範圍的數值時會彈出提示視窗，關閉後修正即可。
      
    * 當使用者按下視窗右上角的「X」，會彈出確認視窗，避免使用者誤按，若沒有要關閉程式按「否(N)」即可返回。
